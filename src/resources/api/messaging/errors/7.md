---
title: "Potential SMS flooding attack"
---
## Error description
This error appears when you try to send multiple identical SMS messages to the same phone number.

By default, you can send up to **5 identical** SMS messages or **20 messages with the same Sender ID** to the same destination number within **60 minutes**.

## How to fix it
If you need to remove this limit, contact support@wavix.com. Include the following details:
- Sender ID
- Destination number
- Date and time you've faced the error
