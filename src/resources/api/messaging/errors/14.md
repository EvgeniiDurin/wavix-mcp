---
title: "Unsupported media format"
---
## Error description
You're trying to send an MMS message with a media file format that isn't supported.

## How to fix it
Make sure your media file matches one of the supported content types and uses the correct file extension.
| Content type | File extension                           |
| --------     | ---------------------------------------- |
| Image        | `jpg`, `jpeg`, `png`, `bmp`              |
| Audio        | `wav`, `mp3`, `amr`, `aac`, `midi`, `mid`|
| Animation    | `gif`                                    |
| Video        | `3gpp`                                   |
| vCard        | `vcf`                                    |
