---
title: "The Sender ID is not provisioned for the country"
---
## Error description
This error appears when you try to send an SMS to a country that isn't approved for your SMS Sender ID.

## How to fix it
Ensure the destination country is allowed for your Sender ID. You can:
- Use the Wavix API to check which countries your Sender ID is approved for
- View the list of allowed destinations in the Wavix Customer portal

<Note>
The time it takes to allow-list the Sender ID varies depending on the destination country. 
</Note>
