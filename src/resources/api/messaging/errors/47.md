---
title: "Messaging to the destination number is blocked for the next 60 minutes"
---
## Error description
You've sent too many messages to the same phone number in a short period. By default, you can send up to **5 identical** SMS messages to the same destination number within **60 minutes**.

## How to fix it
Wait 60 minutes before sending the message again.

If it's urgent, contact support@wavix.com to request an unblock. Include:
- Sender ID
- Destination number
- The date and time of the message
