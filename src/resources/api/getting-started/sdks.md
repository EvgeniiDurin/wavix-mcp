## Wavix Embeddable

Wavix Embeddable is a flexible widget that enables voice calling capabilities in your application or website. With a simple integration, you can add inbound and outbound voice features for your users.

### Key features

- **Voice calling**  
  Support for both inbound and outbound calls directly from your app or CRM.

- **Customizable experience**  
  Tailor the widget's appearance and behavior to match your brand and workflow.

- **Call history**  
  View and manage recent calls within the widget.

- **Webhook events**  
  Receive real-time notifications for call events to trigger workflows or update your UI.

<Tip>
  Wavix Embeddable includes core Wavix features such as Answering Machine Detection (AMD) and voice recording. You can enable these features out of the box. No extra setup required.
</Tip>

### Why use Wavix Embeddable?

Wavix Embeddable makes it easy to add voice calling to your app or website. You can quickly set up inbound and outbound calls, customize how it looks, and view call history. It also supports real-time updates with webhooks and extra features like call recording and answering machine detection.

> [Learn more and get started with Wavix Embeddable](https://github.com/Wavix/wavix-embeddable)
