# Two-Factor Authentication (2FA)

Secure verification service for user authentication

## Features

### SMS Verification

Send one-time passwords via SMS for secure user authentication and transaction confirmation.

### Voice Verification

Deliver OTP codes via automated voice calls for users who prefer or require voice delivery.

### Global Delivery

Reach users worldwide with reliable OTP delivery across all carriers and countries.

### Customizable Templates

Brand your verification messages with custom text and sender IDs.

### Fraud Prevention

Built-in rate limiting and fraud detection to prevent abuse and protect your users.

### Simple Integration

RESTful API with SDKs for quick implementation. Generate, send, and verify codes with minimal code.
