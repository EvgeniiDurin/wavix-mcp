# Campaign Builder

Visual tool for creating voice and SMS campaigns

## Features

### Voice Polls

Create interactive voice surveys with IVR menus and response collection.

### Emergency Broadcasts

Send urgent notifications to large contact lists with priority delivery.

### Marketing Campaigns

Design multi-touch campaigns combining voice and SMS touchpoints.

### Appointment Reminders

Automate reminder calls and texts with confirmation options.

### AMD Integration

Automatic answering machine detection with voicemail drop capabilities.

### Template Library

Pre-built campaign templates for common use cases.

### Number Validator

Validate phone numbers before campaigns to improve delivery rates.

### IVR Builder

Visual drag-and-drop IVR designer for complex call flows.

### Timezone Adjustment

Automatic scheduling based on recipient local time.
