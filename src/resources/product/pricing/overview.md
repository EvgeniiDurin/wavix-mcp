# Wavix Pricing Overview

Simple, transparent pricing with no hidden fees.

## Quick Reference

| Service | Starting Price |
|---------|----------------|
| Inbound Voice | $0.00/min |
| Outbound Voice | $0.004/min |
| Inbound SMS | $0.0075/segment |
| Outbound SMS | $0.0075/segment |
| US Local Number | $1.00/month |
| US Toll-free Number | $2.00/month |

## Plans

### Wavix Flex (Pay-as-you-go)
- No monthly minimum
- Pay only for what you use
- Per-minute billing for voice
- Perfect for testing and variable usage

### Wavix Flex Pro (Volume Discounts)
- $250/month minimum commitment
- Per-second billing (where available)
- E911 support
- Number porting
- Volume discounts
