# SMS FAQ

## How do I send an SMS?

Multiple options available:
1. Send test SMS from your dashboard
2. Use the visual GUI to build messaging flows
3. Run CURL requests from command line
4. Use our REST API programmatically

Check docs.wavix.com for API reference and code examples.

## Do you support SMS campaigns?

Yes. SMS Broadcast is built to power your campaigns. Upload lists, create schedules, build templates, and set filters easily. Contact sales@wavix.com to get started.

## What are the opt-in/opt-out keywords?

Standard opt-out keywords: STOP, UNSUBSCRIBE, CANCEL, END, QUIT
Standard opt-in keywords: START, YES, UNSTOP

These can be customized based on your needs. Opt-out handling is automatic for compliance.

## What is 10DLC?

10DLC (10-Digit Long Code) is the standard for A2P messaging in the US. You must register your Brand and Campaign with The Campaign Registry (TCR) to send messages using US local numbers. See our 10DLC guide for registration steps.
